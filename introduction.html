Introduction to APB

APB was originally developed by James Straughon aka. Athefre. It stands for Athefre's Pair and Block.

Steps

Standard APB consists of 5 steps:

  1. 2x2x3: Solve a 2x2x3 block in the bottom-left. This is usually accomplished by building a Roux-style 1x2x3 block and then extending it by solving the DF and DB edges.
  2. dBR Pair: The back-right F2L pair is solved intuitively.
  3. EO: The edge-orientation of the remaining edges is solved algorithmically (11 algorithms).
  4. LXS: The remaining three F2L pieces are solved algorithmically (116 algorithms).
  5. ZBLL: The last layer is solved (493 algorithms).

This resolves to a hefty number of 620 algorithms. This number can be reduced to 39, by solving LXS intuitively (for which you do need to know the 20 F2L last slot cases) and using OCLL + PLL for the last layer.
